---
title: Getting Started
---

Use this guide to align teams quickly and start using Kairos with clear ownership.

## For marketers

1. Define campaign outcomes and target segments.
2. Confirm available customer attributes and event streams.
3. Start with one activation channel and measure lift.

## For product owners

1. Define lifecycle stages and success metrics.
2. Align required data contracts with engineering.
3. Prioritize the first journey that ties to business KPIs.

## For developers

1. Connect first-party event sources and transactional systems.
2. Validate identity stitching and profile completeness.
3. Expose segment membership and profile data to activation channels.

## Suggested first rollout

- Start with one high-value segment.
- Activate on one channel.
- Measure conversion or retention impact.
- Expand to multi-step journeys after baseline success.
